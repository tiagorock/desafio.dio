Tags

A tags no HTML são elementos usados para estruturar e marcar conteúdos em uma página da web.

Alguns exemplos de tags.

As tags h1, h2, h3, h4, h5, h6, são tags usadas para hierarquias de cabeçalhos, estrutura e organização de páginas da web.

A tag strong é uma tag usada para para destacar um texto importante.

A tag blockquote é usada para representar uma citação no texto.

A tag ol é usada para criar uma lista ordenada onde os itens são ordenados sequencialmente

A tag a é usada para criar um link para outras páginas web.

A tag sub é usada para formatar o texto como subscrito onde o texto é posicionado um pouco abaixo da linha normal.

A tag sup é usada para formatar o texto como sobrescrito, onde o texto é posicionado um pouco acima da linha normal.

A tag hr é usada para criar uma linha horizontal geralmente usada para separar seções de conteúdo.

A tag ul é usada para criar uma lista não ordenada.

A tag font é usada para definir a fonte, tamanho e a cor do texto.

A tag del é usada para representar texto deletado ou riscado.

A tag p é usada para criar parágrafos.

A tag abbr é usada para definir uma abreviação no texto e também é usada para fornecer uma descrição mais detalhada quando passar o mouse por cima do texto

Estes são apenas alguns exemplos de tags.
